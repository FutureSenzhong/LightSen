## 常用快捷键 

### 常用快捷键的查询和配置

- Ctrl + D　　：复制当前行
- Ctrl + E　　： 删除当前行
- Shift + Enter：快速换行
- Ctrl + /　　　: 快速注释（选中多行可以进行批量注释哟）
- Tab　　　　  : 缩进当前行（选中多行后可以批量缩进,取消缩进类似。）
- Shift + tab　  : 取消缩进
- Ctrl + F　　　: 查找
- Ctrl + H　　   : 替换
- Ctrl + 减号　　：折叠当前代码块
- Ctrl + Shift + 减号 ：折叠当前文件
